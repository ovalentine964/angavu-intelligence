📈 Soko Pulse — Market Intelligence

GET /v1/soko/prices

Get current market prices for common goods across informal markets.

Parameters
market    — Market name (e.g., "gikomba", "toi")
category  — Item category (e.g., "vegetables", "grains")
date      — Date (YYYY-MM-DD, default: today)
GET /v1/soko/trends

Get price trends and movement patterns for a specific commodity.

📊 Alama Score — Financial Readiness

POST /v1/alama/assess

Submit transaction data for financial readiness assessment. Uses Bayesian inference to build credit readiness without formal credit history.

Request Body
{
  "transactions": [...],
  "business_type": "retail",
  "duration_months": 12
}

🏛️ Angavu Pulse — MSME Activity Index

GET /v1/pulse/index

Get aggregated MSME activity index for a region. Data is anonymized and differentially private.

Parameters
region   — Region code (e.g., "ke-47" for Migori)
period   — "daily", "weekly", "monthly"

🧠 Federated Learning

POST /v1/federated/submit

Submit encrypted gradient deltas from on-device training. Uses differential privacy (ε=0.1, δ=10⁻⁵) and Byzantine-robust aggregation.

GET /v1/federated/model

Get latest global model version and download URL.

Authentication

API Authentication

API Key

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

API keys are issued per-partner. Contact partners@angavuintelligence.com to request access.

Rate Limits

  • Free tier: 100 requests/hour
  • Partner tier: 10,000 requests/hour
  • Enterprise: Custom limits

Rate limit headers are included in every response:

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1690176000
